# What is Gatsby.js?

<Appear>
  <p>⚛️ Framework for building React apps and websites</p>
  <p>💼 Allows you to bring your own data (i.e. Markdown, headless CMS, Databases, etc.) - JAMstack</p>
  <p>🔌 Plugin ecosystem (+700)</p>
  <p>🔥 Blazing fast, modern JS/CSS, code-splitting, data-prefetching</p>
</Appear>
